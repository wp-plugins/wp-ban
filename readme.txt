Ban users by IP, host name and referer url from visiting your WordPress's blog.

== Description ==

It will display a custom ban message when the banned IP, host name or referer url trys to visit you blog. You can also exclude certain IPs from being banned. There will be statistics recordered on how many times they attemp to visit your blog. It allows wildcard matching too.
All the information (general, changelog, installation, upgrade, usage) you need about this plugin can be found here: [WP-Ban Readme](http://www.lesterchan.net/wordpress/readme/wp-ban.html "WP-Ban Readme").
It is the exact same readme.html is included in the zip package.
